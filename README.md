# docker-vue.js babel

This is a starter project for a node + pug that uses docker for dev enironment.
Docker and docker-compose is all you need to develop and build.

## Stack
* Libsass
* Node
* Pug
* Http-server
* babel
* webpack

## get started
```
$ docker-compose up
```

## gulp
```
$ docker exec -it miproject_web_1 gulp watch
```
